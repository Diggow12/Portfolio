O Desafio

O cliente possuía dados espalhados em diferentes planilhas e sistemas ERP, o que levava a uma demora de 5 dias para gerar um relatório de fechamento mensal.

A necessidade era uma visão consolidada que permitisse decisões baseadas em dados em tempo real, focando em margem de lucro e performance por região.

A Solução

Desenvolvi uma pipeline de dados utilizando SQL Server para centralização e Power BI para visualização. Implementei medidas em DAX complexas para cálculo de tendências (YoY) e Python para limpeza automatizada de dados inconsistentes.

O design seguiu os princípios de UI/UX para dashboards, garantindo que os gestores encontrassem os KPIs principais em menos de 3 segundos.
